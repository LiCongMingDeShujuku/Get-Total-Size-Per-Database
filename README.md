## English
Here’s some very simple sql logic to give you the total size of each database.

---
## Logic
```SQL
;with dbsize as
(
	select
	'database' 		= db_name(database_id)
, 	'size in gb' 	= convert(decimal(10,2),(sum(size * 8.00) / 1024.00 / 1024.00)) 
from
	sys.master_files
group by
	database_id
)
 
select
	*
from
	dbsize
order by
	[size in gb] desc


```
